🔑 Anthropic API 키 설정

AI 운영전략 분석을 사용하려면 Anthropic API 키가 필요합니다.
키는 이 기기의 브라우저에만 저장되며 외부로 전송되지 않습니다.
🔗 API 키 발급 방법
1.console.anthropic.com 접속 후 로그인
2.좌측 메뉴 API Keys 클릭
3.Create Key 버튼 클릭 → 키 복사
4.아래 입력창에 붙여넣기 후 저장
